== Description ==

Like Schema Review plugin? Consider leaving a [5 star review](https://wordpress.org/support/plugin/schema-review/reviews/).

Schema Review is an extension for the Schema plugin, it extend Schema functionality by adding schema.org review and rating Structured Data functionality for Editors and Authors.

The Schema Review extension includes support for [Embedded Review](https://developers.google.com/search/docs/data-types/reviews#review-snippet-examples), which meant to work for Authors and Editorial reviews, editors can review about anything.

This extension require [Schema plugin](https://wordpress.org/plugins/schema/) to be installed and activated on your site.

**Schema Review Key Features**

* Easy to use, minimal settings.
* Minimal output via shortcode, light-weight no CSS or Scripts loaded in front-end.
* Enable Schema Review for Editor or Author.
* Provide a valid markup, test it in Google Structured Data Testing Tool.
* Extend Schema output in JSON-LD format, the most recommended by Google.
* If your site is qualified, Google may display Rich Snippets including rating stars in search results.

Check out the [Plugin Homepage](http://schema.press/) for more info and [documentation](http://schema.press/docs/).

== Installation ==

1. Upload the entire `schema-review` folder to the `/wp-content/plugins/` directory.
2. DO NOT change the name of the `schema-review` folder.
3. Activate the plugin through the 'Plugins' menu in WordPress.
5. In the plugin settings, go to `Schema > Types` and edit any type to enable reviews. 
4. That is all.
